# Heli Frozen Yogurt landing page

This is a simple static landing page for Heli Frozen Yogurt. It is ready for GitHub Pages, Netlify, Vercel, or any static host.

## Files

- `index.html`: the full landing page with embedded CSS
- `assets/heli-logo.png`: trimmed transparent Heli logo
- `assets/heli-stripes.png`: Heli striped background

## Quick GitHub Pages deploy

1. Create a new GitHub repository.
2. Upload `index.html` and the full `assets` folder to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save, then wait for GitHub Pages to publish the site.

## Edit contact email

The CTA buttons currently open an email to `hello@helifrozenyogurt.com`. To change it, search for that email address inside `index.html` and replace it.
